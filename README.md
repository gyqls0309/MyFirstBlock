# MyFirstBlock
  - 택시의 고급화
  
## 현재 문제점
  - 택시를 타고 싶은데 택시를 콜 하기 쉽지 않을 때가 있다.
  - 카카오 택시의 경우 1000 원을 더 내고 콜을 부를 수 있지만, 택시기사에게 400 원이 남는다.
    때문에 오히려 일부러 1000 원을 더 낸 콜은 택시기사들이 잡지 않는다.
  
## 해결 방안
  - 사용자는 기존의 카카오 택시처럼 콜을 부른다.
  - 급한 경우 돈을 더 결제하고 택시를 부른다.
  - 택시는 카카오 택시보다 더 많은 보상을 받는다.
